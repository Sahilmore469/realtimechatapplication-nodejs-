# realtimechatapplication-nodejs-
This is a real-time chat web application developed using Node.js and Socket.io. It allows multiple users to join a common chat room, exchange messages, and get instant updates when someone joins or leaves. The project uses a clean, minimal UI with message bubbles and real-time WebSocket communication.
